# App Store Release Notes: Version 2026.4.0

## What's New in SpeedCube AR

This update is all about precision and advanced techniques! We've finalized the Petrus Method, added a new algorithm library for advanced users, and polished our existing guides.

### 🧩 NEW: Finalized Petrus Method
Master the cube with the **Petrus Method**! Our solver is now 100% reliable and features a detailed, step-by-step interactive guide.
- **Block Building Mastery**: Learn how to build a 2x2x2 and 2x2x3 block efficiently.
- **Edge Orientation**: Learn how to orient all remaining edges at once to simplify your final solve.
- **Visual Clarity**: Each stage of the Petrus method is now illustrated with precise sticker highlighting and 3D demonstrations.

### 📚 NEW: Winter Variation (WV) Library
For the speedcubers looking to shave seconds off their solve, we've added the **Winter Variation Algorithm Library** (Premium).
- **Advanced OLL**: Learn 27 specialized algorithms to solve the yellow cross and top face orientation in a single step during F2L.
- **Interactive Practice**: Each algorithm comes with an interactive 3D demo to help you master the finger tricks.

### 🎓 Tutorial Refinements
- **ZZ Method EOLine**: We've simplified the instructions for the EOLine stage to make identifying "bad edges" more intuitive for beginners.
- **Roux Polish**: Improved the visual sequencing for M-slice moves and block-building in the Roux guide.
- **Premium Clarity**: Corrected our premium upsell info—the Layer-by-Layer (LBL) method remains free for everyone!

### ⚡ Performance & Stability
- Significantly improved solver reliability across all methods (Petrus, ZZ, Roux).
- Optimized search budgets to ensure fast, crash-free solving on all devices.
- Fixed minor UI bugs and camera transition issues.
